## 摘要
图像内机器翻译（IIMT）旨在翻译图像中的场景文本，并将翻译后的文本渲染回原始区域，同时保持整体视觉外观。最近，统一的多模态模型通过在单一框架内结合视觉文本理解和图像生成，提供了一种有前景的解决方案。然而，直接将这些模型应用于IIMT仍然具有挑战性。特别是，它们常常面临理解-生成冲突（即理解阶段推断的翻译与生成阶段使用的文本监督不一致）以及空间位置错位（即渲染的文本未能准确匹配目标文本区域）的问题。为解决这些问题，我们提出了UniTranslator，一个用于IIMT的统一多模态框架，该框架紧密耦合了翻译理解和文本编辑。具体来说，我们引入了一个理解-生成对齐模块（UGAM），以弥合理解和生成之间的表示差距，促进翻译内容预测与文本渲染之间的语义一致性。我们进一步提出了一个空间掩码解码器（SMD），通过对文本区域的像素级监督，改善生成过程中的空间定位、几何对齐和布局可控性。在多个基准上的大量实验表明，UniTranslator在多样化的语言方向和复杂的真实世界布局中均取得了最先进的性能。此外，我们的结果揭示了翻译理解与图像生成之间存在强大的相互增强效应，凸显了统一翻译多模态学习的优势。代码可在https://github.com/SeerRay-Lab/Unitranslator获取。

## Abstract
In-Image Machine Translation (IIMT) aims to translate scene text in an image and render the translated text back into the original regions while preserving the overall visual appearance. Recent unified multimodal models provide a promising solution by combining visual-text understanding and image generation within a single framework. However, directly adapting such models to IIMT remains challenging. In particular, they often suffer from understanding-generation conflicts, where the translation inferred during understanding is inconsistent with the text supervision used in generation, and spatial position misalignment, where the rendered text does not accurately match the target text regions. To address these issues, we present UniTranslator, a unified multimodal framework for IIMT that tightly couples translation understanding and text editing. Specifically, we introduce an Understand-Generation Alignment Module (UGAM) to bridge the representation gap between understanding and generation, encouraging semantic consistency between translated content prediction and text rendering. We further propose a Spatial Mask Decoder (SMD) with pixel-level supervision over text regions to improve spatial grounding, geometric alignment, and layout controllability during generation. Extensive experiments on multiple benchmarks demonstrate that UniTranslator achieves state-of-the-art performance across diverse language directions and complex real-world layouts. Moreover, our results reveal a strong mutual reinforcement effect between translation understanding and image generation, highlighting the advantage of unified translation multimodal learning. Code is available at https://github.com/SeerRay-Lab/Unitranslator.

## 1. 核心问题与整体含义（研究动机和背景）

- **任务定义**：图像内机器翻译（IIMT）要求将图像中的场景文本翻译为目标语言，并将译文写回原文本区域，同时保持背景、布局和视觉风格不变。该技术对于旅行辅助、增强现实、跨境电商等多语言视觉通信场景至关重要。
- **现有方法缺陷**：
  - 传统级联OCR–MT–Editing流水线存在错误累积，泛化性弱。
  - 统一多模态模型（如GPT-Image-1、Qwen-Image）虽然提供了一体化方案，但直接应用时面临两大瓶颈：
    - **理解-生成冲突**：理解分支预测的翻译（语义正确但可能与参考译文不同）与生成分支需要精确匹配某个具体视觉文本实例的监督信号不一致，导致训练矛盾。
    - **空间位置错位**：模型缺乏精细的空间控制，渲染文本可能偏离原文本区域，或导致背景纹理改变。
- **本文目标**：设计一个紧密耦合翻译理解与图像生成的统一框架，同时解决语义一致性和几何对齐问题。

## 2. 方法论：核心思想与关键技术细节

### 2.1 整体架构
UniTranslator 基于 Qwen2.5VL 3B 作为多模态骨干，耦合以下三个分支：
- **理解分支**：自回归预测译文文本 `T_pred`。
- **生成分支**：基于 MMDiT（扩散 Transformer）合成翻译后的图像。
- **掩码分支**：预测文本区域掩码 `M_pred`，提供像素级空间监督。

### 2.2 理解-生成对齐模块（UGAM）
- **目的**：解决理解与生成之间的语义冲突。
- **实现**：以理解表示 `E_und` 为 Query，以多模态表示 `E_VL` 为 Key/Value，通过交叉注意力 + MLP 得到对齐的条件嵌入 `E_c`：
  ```
  E_c = MLP(CrossAttn(E_und, E_VL, E_VL))
  ```
- **作用**：`E_c` 作为生成分支的软条件，隐式传递语义一致的翻译信息，避免硬性文本指令造成的惩罚性梯度冲突。

### 2.3 空间掩码解码器（SMD）
- **目的**：提升文本渲染的空间精度，实现定位、几何对齐和布局保持。
- **实现**：以 `E_und` 为 Query，视觉表示 `E_V` 为 Key/Value，经交叉注意力后用反卷积层解码出掩码预测 `M_pred`。
- **损失**：`L_mask = λ_bce * BCE(M_pred, M_gt) + λ_dice * Dice(M_pred, M_gt)`

### 2.4 条件图像生成
- 生成分支采用扩散 Transformer，以 `E_c` 和文本编码 `E_text` 为条件，预测去噪目标 `v_t`：
  ```
  L_gen = E[ ||v_θ(x_t, t, E_c, E_text) - v_t||^2 ]
  ```
- 其中 `x_t` 是参考图像 `I_tgt` 的噪声潜变量。

### 2.5 两阶段训练策略
- **Stage 1（模块预热）**：冻结 Qwen2.5VL 和扩散主干，仅训练 UGAM 和 SMD。学习率 1e-4，批量 32，训练 70 轮。损失权重 `λ_und=0.1, λ_gen=1, λ_mask=0.1`。
- **Stage 2（联合微调）**：解冻 Qwen2.5VL 的 LoRA（rank=64）、UGAM、SMD 和 MMDiT 注意力投影层。学习率峰值 1e-5（余弦退火），批量 8，100 预热步，训练 30 轮。
- **推理**：训练时 UGAM 使用真实译文 `T_gt`；推理时使用自回归预测的 `T_pred`，第二阶段通过分布差距减少提升鲁棒性。
