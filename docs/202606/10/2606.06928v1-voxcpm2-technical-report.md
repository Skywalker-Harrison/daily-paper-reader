## 摘要
我们提出VoxCPM2，一个完全开源的多语言可控语音生成基础模型，它扩展了VoxCPM的分层扩散-自回归建模范式。VoxCPM2在三个关键维度上推进了该框架：(i)能力方面，通过在一个单一骨干中统一30种语言、9种汉语方言、自然语言语音设计、风格可控语音克隆以及高保真续接克隆；(ii)质量方面，通过非对称AudioVAE以16kHz编码、48kHz重构，实现高编码效率的隐式超分辨率；(iii)规模方面，通过将模型参数联合扩展到2B，训练数据超过200万小时的多语言语音。为了在一个模型内支持这些多样化的能力，我们引入了一种统一的序列组织方式，通过相同输入构建块的不同排列来表达所有生成模式，从而在单一参数集和目标下进行联合训练。VoxCPM2在公开的零样本和指令跟随TTS基准上达到了最先进或具有竞争力的性能。在我们内部的30语言评估集上，它取得了1.68%的平均WER。这些结果表明，层次化连续潜变量建模无需依赖任何外部离散语音分词器，为大规模多语言和可控语音生成提供了一种可行且强大的基础。模型权重、微调代码和推理工具在Apache 2.0许可下公开发布，以促进社区研究和开发。

## Abstract
We present VoxCPM2, a https://info.arxiv.org/help/prep#abstractsfully open-source multilingual and controllable speech generation foundation model that extends the hierarchical diffusion-autoregressive modeling paradigm of VoxCPM. VoxCPM2 advances the framework in three key dimensions: (i) capability, by unifying 30 languages, 9 Chinese dialects, natural-language voice design, style-controllable voice cloning, and high-fidelity continuation cloning within a single backbone; (ii) quality, through an asymmetric AudioVAE that encodes at 16 kHz and reconstructs at 48 kHz, enabling implicit super-resolution with high encoding efficiency; and (iii) scale, by jointly scaling the model to 2B parameters and the training data to over 2 million hours of multilingual speech. To support these diverse capabilities within one model, we introduce a unified sequence organization that expresses all generation modes through different arrangements of the same input building blocks, allowing joint training under a single set of parameters and objective. VoxCPM2 achieves state-of-the-art or competitive performance on public zero-shot and instruction-following TTS benchmarks. On our internal 30-language evaluation set, it attains an average WER of 1.68%. These results demonstrate that hierarchical continuous-latent modeling, without relying on any external discrete speech tokenizer, offers a viable and powerful foundation for large-scale multilingual and controllable speech generation. The model weights, fine-tuning code, and inference tools are publicly released under the Apache 2.0 license to foster community research and development.

## 1. 论文的核心问题与整体含义（研究动机和背景）

- **研究动机**：现有文本到语音（TTS）系统虽然已能生成自然语音，但在多语言覆盖、高保真音频、以及自然语言控制的灵活性上仍存在瓶颈。主流方法（如 VALL-E、CosyVoice）采用离散语音分词器（如 RVQ）进行序列建模，虽然利用了 LLM 的扩展优势，但量化必然丢失精细声学细节；多阶段流水线（AR + 扩散/流匹配）虽能提升质量，却割裂了语义规划与声学渲染，无法端到端联合优化。
- **整体含义**：VoxCPM2 旨在扩展层次连续潜变量建模范式（源自 VoxCPM），构建一个 **全开源、多语言、可控** 的语音生成基础模型。它不依赖任何外部离散语音分词器，直接在连续潜在空间作业，统一 30 种语言、9 种中文方言，并支持自然语言语音设计、可控语音克隆和续接克隆等多种模式。论文论证了层次连续潜变量建模可以作为大规模多语言可控 TTS 的可行且强大的基础。

## 2. 论文提出的方法论

### 核心思想
- 保留 VoxCPM 的 **层次扩散自回归** 架构：Text-Semantic Language Model (TSLM) 捕捉高层语义/韵律 → Finite Scalar Quantization (FSQ) 可微分半离散瓶颈 → Residual Acoustic Language Model (RALM) 恢复声学细节 → Local Diffusion Transformer (LocDiT) 生成连续潜变量 patch。
- 所有生成模式（基本 TTS、语音设计、参考克隆、可控克隆、续接克隆）通过 **统一的序列组织** 实现，即仅改变输入构件的排列，骨干参数和目标函数完全共享。

### 关键技术细节
- **AudioVAE V2**：非对称变分自编码器，**16 kHz 编码 → 48 kHz 解码**，保持紧凑的 6.25 Hz 自回归序列 token 率（patch size = 4）。编码器用 16 kHz 输入可复用大规模 16 kHz 语料并避免采样率不匹配；解码器提升至 48 kHz 改善波形保真度，实现隐式超分。
- **骨干网络改进**：
  - 用 **拼接-投影融合** 替代 VoxCPM 中的求和融合，增强 RALM 输入信息流。
  - **多 token 条件前缀**：LocDiT 接收分离的语义、残差和扩散时间步 embedding 作为独立 prefix token，避免信息过早坍缩。
  - FSQ 瓶颈维度从 256 增至 512（9 级每维）。
  - 移除 RALM 中的 RoPE 位置编码（NoPE 设计），提升长句稳定性。
- **参考音频路径**：引入可选的独立参考音频段（`<REF_START>`...`<REF_END>`），无需转录文本即可提供说话人身份信息。
- **统一序列组织**：五种配置仅通过输入布局区分（参见论文表 2）。
- **三阶段渐进课程训练**：
  1. **多语言 TTS 预训练**：基础 `(文本, 音频)` 对，最大序列长度 4096。
  2. **联合 TTS + 可控 TTS 预训练**：逐步引入可控数据，序列长度扩展至 8192。
  3. **高质量退火 SFT**：使用精标可控数据，平衡语言采样，学习率退火。
- **数据构建**：通过情感分类预筛选、多维度自然语言标注（年龄、性别、口音、情感等）、相同说话人参考挖掘、以及“克隆合成”解耦风格与内容（用模型自身在无关文本上克隆原有风格，减少风格-文本相关性）。

### 公式 / 算法流程（文字说明）
生成第 \(i\) 个 patch 的流程：
1. 通过 **Local Encoder** 从历史潜变量 \(z_{<i}\) 得到 patch 级声学嵌入 \(E_{<i}\)。
2. **TSLM** 以文本 token \(T\) 和 \(E_{<i}\) 为输入，输出高层隐藏状态；再经 **FSQ** 层输出半离散语义骨架 \(h_i^{\mathrm{FSQ}}\)。
3. **RALM** 融合文本隐藏状态、\(h_i^{\mathrm{FSQ}}\) 和 \(E_{<i}\)，输出残差声学隐藏 \(h_i^{\mathrm{residual}}\)。
4. **LocDiT** 接收 \(h_i^{\mathrm{FSQ}}\)、\(h_i^{\mathrm{residual}}\)、扩散时间步 \(t\) 以及前一个 patch 和当前噪声 patch，通过流匹配预测速度场，生成当前 patch 的清晰潜变量。
5. 停止预测器基于 FSQ 隐藏状态决定生成终止。

1. **VoxCPM2 在公共零样本克隆、多语言合成、指令跟随等任务上达到 SOTA 或竞争性水平**。在 Seed-TTS-Eval 上，test-EN WER 1.84%/SIM 75.3，test-ZH CER 0.97%/SIM 79.5。在 InstructTTSEval 英文上以 84.2/83.2/71.4 领先。

5. **训练数据规模（超 200 万小时）和模型参数（2B）的联合缩放显著提升了多语言覆盖和智能化**，内部 30 语言平均 WER 仅 1.68%。

## 7. 优点
