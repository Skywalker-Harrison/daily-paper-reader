## 摘要
自动语音识别（ASR）已成为人机交互的关键技术。然而，由于跨不同语言对的多语言代码切换语音资源严重匮乏，代码切换ASR（CS-ASR）仍然特别具有挑战性。现有方法主要通过合成CS语音生成或在有限的双语数据集上进行特定语言对的微调来提高CS-ASR性能。然而，这些方法面临固有的可扩展性限制，因为对CS的支持必须针对不同语言对分别开发，而语言对的数量随支持的语言数量呈组合增长。在这项工作中，我们探究了从有限的所见语言对中学到的CS能力是否能通过模型合并和领域泛化方法泛化到未见过的语言对。我们的实验表明，合并的双语CS-ASR模型能适度泛化到未见过的语言对，这表明双语CS能力跨语言对的迁移有限。

## Abstract
Automatic Speech Recognition (ASR) has become a key technology for human--AI interaction. However, code-switching ASR (CS-ASR) remains particularly challenging due to the severe scarcity of multilingual CS speech resources across diverse language pairs. Existing approaches primarily improve CS-ASR performance through synthetic CS speech generation or pair-specific fine-tuning on limited bilingual datasets. Nevertheless, these approaches face an inherent scalability limitation, as support for CS must be developed separately for language pairs whose number grows combinatorially with the number of supported languages. In this work, we investigate whether CS capabilities learned from a limited set of seen language pairs can generalize to unseen language pairs through model merging and domain generalization methods. Our experiments show that merged bilingual CS-ASR models modestly generalize to unseen language pairs, suggesting limited transfer of bilingual CS capabilities across language pairs.

- **现有方法局限**：已有工作通过合成CS语音或针对特定语言对微调来提升性能，但这些方法扩展性差，无法高效支持未见过的语言对。

## 2. 方法论
### 核心思想
- 利用Whisper-Medium作为骨干模型，先在三个已见双语CS数据上分别微调，获得针对性模型；然后通过**模型合并**或**领域泛化**训练方式，尝试将已见对的能力迁移到未见对，无需使用未见对的数据。

### 关键技术细节
1. **模型合并（Model Merging）**：
   - 方法：将不同语言对微调后的模型参数进行合并。
   - 具体算法：
     - **Task Arithmetic**：简单线性组合任务参数差值。
     - **TIES-Merging**：通过稀疏符号一致性解决参数冲突。
     - **DARE**：随机剪枝与缩放提高合并鲁棒性。
   - 实现工具：MergeKit。

2. **领域泛化（Domain Generalization）**：
   - 方法：在多个已见对的数据上联合训练，使模型学习跨领域不变特征。
   - 具体算法：
     - **Fish**：最大化梯度一致性。
     - **Fishr**：对齐域级别梯度方差。
     - **GGA-L**（Gradient-Guided Annealing的轻量版）：通过早期梯度对齐缓解领域过拟合。

3. **参数分析**：
   - 通过层间行级平均绝对值（MAV）阈值比率，比较微调前后参数变化，发现高层编码器和解码器层更新更大，低层几乎不变，表明CS适应主要发生在语义/语言表征层面。

## 6. 主要结论与发现
1. **单语言对微调可部分迁移**：在已见对微调也能改善其他已见对的MER，但跨未见对的效果弱（如KO-EN微调后KO-JA的MER仅从0.44降到0.46，反而恶化）。
2. **模型合并优于简单微调**：TIES-Merging在已见对和未见对上都表现稳定，优于Task Arithmetic和DARE，尤其2模型合并时在未见对上MER达0.32-0.34。
3. **领域泛化效果有限**：仅Fishr在未见对上平均MER有所改善（0.33 vs 联合微调0.41），但仍远低于实用水平（>0.3）。
4. **CS适应集中在高层**：参数分析显示编码器/解码器高层参数更新大，低层几乎不变，说明CS识别依赖语义级推理，而非底层声学特征。
5. **整体局限**：所有方法在未见对上的最佳MER（0.32）仍远高于已见对微调后的<0.2，表明当前泛化方法不足以支撑真实多语言CS-ASR。

## 8. 不足与局限

- **语言覆盖有限**：未评估完全未见语言的泛化能力（如训练中从未出现的法语、中文等）。
- **领域泛化方法欠适配**：传统DG方法假设输出分布不变，而CS-ASR因语言对变化导致输出空间改变，原生DG方法不适合——作者虽讨论但未设计适配方案。

- **应用限制**：当前最优未见对MER>0.3，无法实际部署；未来需要更专门的方法、更大规模数据和多骨干验证。
