## 摘要
虽然现有的文本到语音（TTS）模型表现出高度的表现力，但由于离散文本意图与连续声学实现之间的结构不匹配，对复合指令的细粒度控制仍然具有挑战性。受人类认知解耦的启发，我们提出了AgentSteerTTS，一种面向复合指令的意图忠实表现控制的多智能体闭环框架。首先，在我们的框架中，一个对抗性解耦智能体通过学习可分离的身份和情感韵律子空间并采用泄露抑制正则化来减轻说话人-情感泄露。接着，一个双流锚定控制器利用大规模声学原型库来锚定抽象意图：检索智能体选择表现性锚点，而合成智能体通过门控注意力将它们融合成连续控制向量。最后，一个快慢反馈智能体通过潜在梯度校正来细化输出强度，并利用高层感知批判来解决语义-声学不匹配。在复合指令基准和公开测试集上的实验表明，AgentSteerTTS对基线模型产生了一致且显著的改进，证明了所提出方法的有效性。

## Abstract
While existing text-to-speech (TTS) models exhibit high expressiveness, fine-grained control over composite instructions remains challenging due to the structural mismatch between discrete textual intents and continuous acoustic realizations. Inspired by human cognitive decoupling, we introduce AgentSteerTTS, a multi-agent closed-loop framework designed for intent-faithful expressive control of composite instructions. First, in our framework, an adversarial disentanglement agent mitigates speaker-emotion leakage by learning separable identity and emotion-prosody subspaces with leakage-suppressing regularization. Next, a Dual-Stream Anchoring Controller grounds abstract intents using a large-scale acoustic prototype library: a Retrieval Agent selects expressive anchors, while a Synthesis Agent fuses them into continuous control vectors via gated attention. Finally, a Fast-Slow Feedback Agent refines output intensity through latent gradient correction and resolves semantic-acoustic mismatches using high-level perceptual critique. Experiments on a composite-instruction benchmark and public test sets show that AgentSteerTTS yields consistent and significant improvements to the baselines, demonstrating the effectiveness of the proposed method.

- **研究动机**：现有文本到语音（TTS）模型虽能生成高自然度的语音，但在**复合指令**（如“开心但略带傲慢”）的细粒度表现控制上严重不足。用户往往需要反复调整提示词和重新采样才能接近目标效果。
- **核心问题**：
  - **离散文本意图与连续声学特征的结构性失配**：复合指令对应的声学控制空间通常是多模态、非凸的，而现有模型采用确定性映射，导致输出趋向于多个模态的平均，造成属性稀释（目标维度抑制、非目标维度泄露）。
  - **说话人-情感纠缠**：说话人音色与情感韵律在声学特征中耦合，优化时会相互牵制，产生负相关（提升情感保真度可能损害说话人相似度）。
- **整体含义**：本文旨在通过多智能体闭环框架，显式对齐高层语义意图与底层声学实现，实现复合指令的**意图忠实**（intent-faithful）表现控制，同时保持说话人身份和自然度。

## 2. 论文提出的方法论：核心思想、关键技术与算法流程（文字说明）

### 2.1 核心思想
受人类认知解耦启发，设计**解耦 → 锚定 → 精炼**的闭环机制，将复合指令逐步转换为精确的声学控制信号。

### 2.2 关键技术细节

#### （a）对抗解耦智能体（Adversarial Disentanglement Module, ADM）
- **目标**：分离说话人身份子空间 \(Z_{id}\) 和情感韵律子空间 \(Z_{emo}\)，减少交叉信息泄漏。
- **实现**：
  - 使用身份编码器 \(E_{id}\) 和情感编码器 \(E_{emo}\) 提取潜变量。
  - 通过**重构损失** \(L_{rec}\) 保证两因子联合解码可恢复梅尔谱。
  - 加入**交叉协方差正交约束** \(L_{orth}\) 惩罚两因子间的相关性。
  - 采用**梯度反转层（GRL）** 进行双向对抗训练：鉴别器 \(D_{id}\) 从情感流预测说话人，\(D_{emo}\) 从身份流预测情感，使 \(Z_{id}\) 情感不变、\(Z_{emo}\) 说话人不变。
- **总损失**：\(L_{ADM} = L_{rec} + \lambda_{adv} L_{adv} + \lambda_{orth} L_{orth}\)。

#### （b）双流锚定控制器（Dual-Stream Anchoring Controller, DAC）
- **目的**：利用大规模声学原型库将抽象意图“锚定”到具体声学向量。
- **流程**：
  1. **检索智能体**：将用户指令改写为多个查询，在原型库中检索最匹配的高表现力音频段 \(x_{pro}\)，并进行一致性裁剪（选出与意图最一致的时窗），得到声学锚点 \(x'_{pro}\)。
  2. **合成智能体**：将锚点音频、说话人参考音频和指令权重 \(w\) 转换为两个控制流：
     - **声学流**：通过情感编码器得到 \(q_{ref}\)（来自锚点）和 \(q_{base}\)（来自说话人中性状态），线性插值得到混合向量 \(q_{mix}\)。
     - **文本流**：根据权重聚合风格原型得到符号嵌入 \(q_{txt}\)。
  3. **自适应融合**：\( \hat{z}_{emo} = q_{txt} + (1 - \sum w_k) \cdot q_{mix} \)，既保留显式意图又补充自然韵律。

#### （c）快慢反馈智能体（Fast-Slow Feedback Agent）
- **快速智能体（Fast Agent）**：
  - 在解码前使用轻量化的可微**潜在一致性预测器（LCP）** \(R_{\phi}\)，以缩放因子 \(\alpha\) 控制 \(\hat{z}_{emo}\) 注入强度。
  - 在线梯度下降更新 \(\alpha^*\)（目标：减小预测嵌入与目标文本嵌入的余弦距离），无需运行声码器，延迟约121ms。
- **慢速智能体（Slow Agent）**：
  - 声码器生成最终波形后，**监督智能体（MLLM）** 提供自然语言批判（如“情感过弱/过强”、“情感类型错误”）。
  - 根据批判类型触发：
    - **强度微调**：更新 \(\alpha\) 并重新调用 Fast Loop。
    - **条件重置**：重新调用 DAC 更新 \(\hat{z}_{emo}\)。

### 2.3 算法流程（文字描述）
1. **预训练阶段**：先预训练基础TTS模型。
2. **联合训练阶段**：同时优化 ADM（\(L_{adv}, L_{rec}\)）和 DAC（通过检索、融合损失）。
3. **推理阶段**：
   - 输入复合指令，DAC 生成初始 \(\hat{z}_{emo}\)。
   - Fast Loop 执行 \(T=2\) 次梯度校正得到 \(\alpha^*\)。
   - 解码+声码器生成波形。
   - Slow Loop 评估波形：
     - 若强度偏差 → 调整 \(\alpha\) 并重跑 Fast Loop。
     - 若类型错误 → 重跑 DAC。

## 6. 论文的主要结论与发现

1. **系统性偏差确认**：复合指令下现有模型存在**目标抑制**（目标情感维度激活降低 25%–45%）和**非目标泄漏**（平均 +0.08），并非随机噪声。
2. **AgentSteerTTS 有效性**：
   - 在 ESD 基准上达到最高情感 MOS（ESMOS 4.42）、自然度 MOS（SNMOS 4.52）、说话人相似 MOS（SSMOS 4.31）。
   - 在复合指令基准上 E-SIM 提升至 0.955（IndexTTS2 为 0.864），S-SIM 提升至 0.841，WER 降至 1.34%。
3. **ADM 缓解纠缠**：去除 ADM 后 S-SIM 下降 0.027，且 E-SIM 与 S-SIM 的相关性从 r=0.544 降至 r=-0.032，证实解耦有效。
4. **原型检索关键**：文本仅条件下 CSR 从 0.78 降至 0.49；感知裁剪进一步降低泄漏。
5. **闭环反馈提升鲁棒性**：去掉 Fast/Slow Loop 后 CSR 分别降至 0.70/0.67，表明它们主要减少失败样本而非提升平均值。

## 7. 优点（方法或实验设计上的亮点）

- **问题定义新颖**：明确揭示复合指令下“目标抑制+非目标泄漏”的结构性偏差，并从多模态可行域角度理论分析。
- **方法设计的仿生学启发**：借鉴“快慢思考”认知理论（Fast & Slow Thinking），结合梯度校正（快速）和语义批判（慢速），实现高效且可靠的闭环精炼。
- **解耦与锚定有机整合**：ADM 先创造干净的因子空间，DAC 再用具体原型“锚定”抽象意图，避免了仅在文本空间中调整的局限性。
- **实验验证充分**：不仅报告数值指标，还通过 t-SNE 可视化纠缠解除、能量分配分析、频谱差异图等提供多维度证据。
- **公平性意识**：在对比中明确指出全系统是检索增强的，并给出了无检索的消融版本以供参考；同时分离 Slow Loop 后端以说明架构增益。

## 8. 不足与局限

- **原型库依赖**：性能受限于原型库的覆盖率和检索可靠性，对于罕见或不寻常的复合风格效果可能下降。
- **ADM 仅解耦主要因子**：只针对说话人-情感耦合，未处理录音条件、口音、年龄等残余变化。
- **Slow Loop 依赖外部评估器**：使用 MLLM（如 Gemini）作为批判后端，引入额外延迟（≤200ms）且可能对分布外提示过于自信或犯错，未能完全自包含。
- **自然度-表现力权衡**：复合控制下 DMOS 低于某些基线（3.82 vs 4.31），证实了“更强属性的韵律可能使声音更突出”的 trade-off。
- **未见扩展验证**：实验仅在英/中文数据集（ESD, MSP-Podcast）上进行，跨语言、跨领域（如歌声、语音转换）的泛化性未知。
- **未提供完整开源**：计划仅发布基准提示/元数据、评估代码和 demo，但完整模型权重受政策限制可能不公开，影响可复现性。
